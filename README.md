# DesignPattern 設計模式

## 簡述

設計模式，工程師們將各類問題統整出來的解決方案

是針對特定的情境及需求的解決方法之一，並不一定是唯一解

設計模式只是一個工具，評估帶來效益及可承受風險才是正確的使用方式
